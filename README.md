# Description

Portfolio is a front-end application, in Angular + UI.

</br>

## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd projet
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng start
```

Run tests

```bash
  ng test
```
